# STEM-Two-Discipline-Connections
This interactive artifact was created for a STEM scholars’ program at the undergraduate level. As part of the instructional and supporting materials for students enrolled in the STEM scholars’ program, this STEM 3 discipline visualization illustrates to scholars the interdisciplinary possibilities of STEM subject areas related to commitment to study and professional careers. This STEM 3 Discipline visualization belongs to a collection showing cross disciplines of STEM 2 discipline, STEAM 2 discipline, STEM 3 and STEAM 3 discipline connections.
